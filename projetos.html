OneRing Chat

Este preview da versão beta de um chat anônimo que está sendo migrada para o framework
Angular com conexão entre salas feitas com WebSocket.
Criei a página inicialmente com puro HTLM5, CSS3 e javascript. Ele gera nome de usuário e
cor de texto aleatória ao entrar. É possível enviar emojis, após o primeiro texto ser enviado,
Assim como a funcionalidade de enviar o texto para a tela só é liberada após se conectar.

Site Da Hora!

Este site simples, mostra a data e hora atuais de acordo com quem acessa e altera

Spider Chase

Animação interativa em canvas onde aranhas brancas perseguem o cursor do mouse com
fluidez. Esse simples projeto usa HTML5, CSS e JavaScript puro e integrado,
demonstrando desenho vetorial e manipulação de eventos.
Há uma surpresinha ao chegar em 50 aranhas :D

Salém The Cat!

Uma das primeiras webpages que criei, a arte e a animação foram criadas por mim
usando Krita. Tem um botão de curtir que armazena as curtidas de cada sessão e além
disso aparecem janelas agradecendo as curtidas com mensagens engraçadas.
